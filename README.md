## 🎯 项目简介

### 它是什么？

这是一个微信小程序，用户在小程序中输入自己的专业、性格、工作偏好等信息，AI会基于招聘市场数据分析你的职业韧性，推荐适合你的管理类岗位，并给出个性化的职业发展规划和能力培养方案。

### 它能做什么？

| 功能 | 说明 |
|------|------|
| 📊 招聘大数据趋势洞察 | 展示岗位需求变化、技能词云、能力雷达图 |
| 🤖 AI韧性评估 | 评估你的抗压能力、适应能力、学习能力等5个维度 |
| 💼 岗位适配推荐 | 基于你的画像匹配最合适的岗位，给出匹配度和理由 |
| 🗺️ 职业发展规划 | 为你设计1年短期和3年中期的职业发展路径 |
| 📚 能力培养方案 | 针对你的能力缺口推荐课程、AI工具实操练习 |

### 技术栈

| 模块 | 技术 |
|------|------|
| 前端 | 微信小程序（原生框架） |
| 后端 | Node.js + Express |
| 数据库 | SQLite（轻量级嵌入式数据库，无需安装） |
| AI引擎 | 通义千问 / 智谱GLM / DeepSeek 大模型API |
| 数据采集 | Python 爬虫 |

## 🤔 为什么不能像微信QQ那样"开箱即用"？

如果你之前只用过微信、QQ、抖音这类软件，可能会疑惑：**为什么这个项目需要启动后端、配置API Key，这么麻烦？**

### 软件分两种

| 类型 | 例子 | 特点 |
|------|------|------|
| **C/S应用**（客户端/服务器） | 微信、QQ、抖音 | 你只装一个App，服务器由腾讯/字节跳动运维，你感知不到后端的存在 |
| **开发项目**（本项目） | 本项目 | 你需要同时运行前端、后端，因为**你就是开发者，你自己在搭建服务器** |

### 打个比方

- **用微信** = 去餐厅吃饭，你只管点菜和吃，厨房（服务器）是餐厅的
- **用本项目** = 自己开餐厅，你得自己建厨房（后端）、买食材（数据库）、请厨师（AI大模型）

### 为什么需要数据库？

微信的聊天记录存在腾讯的服务器上，你不需要关心。但本项目的岗位数据、用户信息、评估结果需要存在**你自己的数据库**里。本项目使用 SQLite 数据库，它是一个轻量级的嵌入式数据库，数据库就是一个文件，无需单独安装数据库服务。

### 为什么需要同时启动前后端？

- **前端**（微信小程序）= 用户看到的界面，负责展示和交互
- **后端**（Node.js服务）= 幕后的逻辑，负责处理数据、调用AI大模型

前端不能直接调用AI大模型（那样API Key就暴露了），必须通过后端中转。所以两者必须同时运行。

### 为什么需要API Key？

AI评估功能依赖大模型（类似ChatGPT的AI），调用大模型需要付费的API Key。就像打电话需要充话费一样，调用AI也需要付费。不过通义千问、智谱GLM和DeepSeek都有免费额度，足够开发测试使用。

### 用 Trae 打开项目

1. 下载并安装 Trae：https://www.trae.ai/
2. 打开 Trae，点击「打开文件夹」
3. 选择本项目的根目录（`wxproject` 文件夹）
4. 左侧会显示项目文件结构
5. 点击任意文件即可编辑
6. 右侧的 AI 对话框可以直接问问题

### 修复微信开发者工具中的Bug

当微信开发者工具控制台出现红色报错时：

**步骤1：复制错误信息**

在微信开发者工具底部的「调试器」→「Console」中，找到红色的错误信息，全选复制。

**步骤2：在Trae中提问**

在 Trae 右侧的 AI 对话框中，粘贴错误信息，例如：

```
微信小程序报错：
TypeError: Cannot read property 'map' of undefined
    at pages/index/index.js:15:20
请帮我修复这个错误
```

**步骤3：按照AI的建议修改**

AI会告诉你需要修改哪个文件的哪行代码，以及怎么改。按照建议修改后保存，微信开发者工具会自动热重载。

**步骤4：验证修复**

回到微信开发者工具，确认红色报错消失，功能正常。

### 常见Bug及修复思路

| Bug现象 | 可能原因 | 修复思路 |
|---------|---------|---------|
| 页面空白 | JS代码报错 | 查看控制台错误，修复对应文件 |
| 网络请求失败 | 后端未启动或地址错误 | 确认后端运行中，检查 `config/index.js` 中的 `baseUrl` |
| 数据不显示 | 接口返回数据格式不匹配 | 检查后端返回的JSON格式，与前端解析逻辑对照 |
| 点击无反应 | 事件绑定错误 | 检查 `bindtap` 对应的函数名是否正确 |
| 样式错乱 | CSS类名拼写错误 | 检查 `class` 属性和 `wxss` 中的类名 |

## 🔄 如何进行功能迭代

### 迭代开发流程

```
1. 明确要做什么功能
   ↓
2. 在Trae中描述需求，让AI帮你写代码
   ↓
3. 修改代码并保存
   ↓
4. 在微信开发者工具中测试
   ↓
5. 发现Bug → 回到Trae修复
   ↓
6. 功能完成 → 更新文档
```

### 如何向AI描述需求

**不好的描述**：
> 加一个功能

**好的描述**：
> 在评估结果页面，增加一个"分享给朋友"按钮，点击后生成一张包含韧性得分和适配岗位的图片，用户可以保存到相册或分享到微信聊天。

**描述需求的模板**：
```
在【哪个页面】，增加一个【什么功能】，用户点击【什么按钮】后，会【发生什么效果】。
```

### 迭代示例

#### 示例1：增加"收藏岗位"功能

在 Trae 中输入：
```
请在评估结果页面的每个岗位卡片右上角，添加一个收藏按钮（心形图标）。
点击后调用后端 /api/user/favorite 接口保存收藏。
在个人中心页面增加"我的收藏"列表，展示收藏的岗位。
后端接口已经设计好了，请帮我写前端代码。
```

#### 示例2：增加"深色模式"

在 Trae 中输入：
```
请为微信小程序添加深色模式支持：
1. 在 app.wxss 中定义深色模式的CSS变量
2. 在个人中心添加深色模式开关
3. 切换时保存到本地存储
4. 所有页面根据开关切换样式
```

## 💰 部署上线的成本分析

### 本项目适合本地开发测试，不建议直接上线

要真正将小程序发布到微信平台供所有人使用，需要以下成本：

| 项目 | 费用 | 说明 |
|------|------|------|
| 微信小程序认证 | 300元/年 | 个人号无法发布，需要企业认证 |
| 域名 | 50-100元/年 | 需要备案（约7-15天） |
| 云服务器 | 500-2000元/年 | 2核4G起步，推荐阿里云/腾讯云 |
| SSL证书 | 免费-5000元/年 | Let's Encrypt免费，商业证书收费 |
| 大模型API | 按量计费 | 通义千问约0.004元/千token，用户量大会很贵 |
| **合计** | **约1000-8000元/年** | 取决于用户量和配置 |

**注意**：使用 SQLite 数据库无需单独的数据库服务器费用，数据库文件直接存储在云服务器上。

### 为什么只能本地使用？

1. **微信小程序上线要求**：
   - 必须使用 HTTPS（需要域名+SSL证书）
   - 服务器域名必须在微信后台配置
   - 必须通过微信审核

2. **API Key 安全问题**：
   - 大模型 API Key 不能暴露在前端
   - 必须通过后端中转调用
   - 需要服务器部署后端

3. **数据合规**：
   - 爬虫采集的数据可能涉及版权
   - 用户数据存储需要符合隐私法规

### 如果想上线，需要做什么？

1. 注册公司或使用已有企业资质
2. 购买域名并完成ICP备案
3. 购买云服务器，部署后端
4. 配置HTTPS和Nginx反向代理
5. 在微信后台配置服务器域名
6. 提交微信审核发布

## ❓ 常见问题

### Q1：我完全不会编程，能用这个项目吗？

**可以！** 但需要花时间学习。建议：
1. 按照环境搭建指南一步步操作
2. 遇到问题复制错误信息，问 Trae/豆包 等 AI 工具
3. 不要害怕报错，报错是正常的，AI能帮你解决大部分问题

### Q2：后端启动后访问 localhost:3000 显示无法访问？

检查以下几点：
1. 后端是否真的启动成功了（看命令行有没有报错）
2. `.env` 文件中的配置是否正确
3. 3000 端口是否被其他程序占用

### Q3：微信开发者工具打开后页面空白？

1. 打开调试器（底部的 Console 面板），看红色报错
2. 确认勾选了「不校验合法域名」
3. 确认后端正在运行
4. 把报错信息复制到 Trae 中让 AI 帮你修

### Q4：AI评估一直转圈不出结果？

1. 检查 `.env` 中的 API Key 是否正确（根据你选择的LLM_PROVIDER）
2. 检查网络是否能访问大模型API
3. 查看后端命令行是否有报错
4. API Key 额度可能用完了

### Q5：如何更新项目代码？

用 Trae 打开项目，在 AI 对话框中描述你想修改的内容，AI 会帮你修改代码。修改后保存，微信开发者工具会自动刷新。

### Q6：我想加一个新功能，但不知道怎么描述？

用最自然的话描述，比如：
- "我想在首页加一个搜索框"
- "我想让评估结果可以保存到手机"
- "我想加一个深色模式"

AI 会帮你拆解成具体的代码修改步骤。

### Q7：SQLite 数据库文件在哪里？

SQLite 数据库文件位于 `backend/data/ai_career.db`，会在后端首次启动时自动创建。可以使用 DB Browser for SQLite 打开查看和编辑数据。
