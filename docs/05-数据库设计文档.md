## 4. 初始化SQL脚本

```sql
PRAGMA journal_mode = WAL;
PRAGMA foreign_keys = ON;

-- 用户表
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    openid TEXT NOT NULL UNIQUE,
    nickname TEXT DEFAULT '',
    avatar TEXT DEFAULT '',
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    updated_at TEXT DEFAULT (datetime('now', 'localtime'))
);

-- 评估记录表
CREATE TABLE IF NOT EXISTS assessments (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    answers TEXT,
    personality_result TEXT,
    interest_result TEXT,
    ability_result TEXT,
    values_result TEXT,
    overall_result TEXT,
    score INTEGER DEFAULT 0,
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);

CREATE INDEX IF NOT EXISTS idx_user_id ON assessments(user_id);

-- 职业规划表
CREATE TABLE IF NOT EXISTS career_plans (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    assessment_id INTEGER,
    career_direction TEXT,
    career_plan TEXT,
    skill_gaps TEXT,
    learning_plan TEXT,
    created_at TEXT DEFAULT (datetime('now', 'localtime')),
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE,
    FOREIGN KEY (assessment_id) REFERENCES assessments(id) ON DELETE SET NULL
);

CREATE INDEX IF NOT EXISTS idx_career_user_id ON career_plans(user_id);
```

## 5. SQLite 特性说明

### 5.1 SQLite 与 MySQL 的主要差异

| 特性 | MySQL | SQLite |
|------|-------|--------|
| 数据类型 | 严格类型系统 | 动态类型（TEXT/NUMERIC/INTEGER/REAL/BLOB） |
| JSON支持 | 原生JSON类型 | 使用TEXT存储，通过json()函数处理 |
| 日期时间 | DATETIME类型 | TEXT类型存储，使用datetime()函数 |
| 自增主键 | AUTO_INCREMENT | AUTOINCREMENT |
| 外键约束 | 默认启用 | 需要PRAGMA foreign_keys = ON |
| 字符集 | utf8mb4 | 默认UTF-8 |
| 事务 | 默认自动提交 | 默认自动提交，支持BEGIN/COMMIT |

### 5.2 SQLite 优势

- **零配置**：无需安装数据库服务，数据库就是一个文件
- **轻量级**：整个数据库引擎只有几百KB
- **跨平台**：支持所有主流操作系统
- **嵌入式**：可直接嵌入应用程序中
- **单文件**：便于备份和迁移

### 5.3 注意事项

1. **并发限制**：SQLite 适合读多写少的场景，不适合高并发写入
2. **数据类型**：SQLite 使用动态类型，需要应用层进行类型检查
3. **JSON处理**：需要使用 json() 函数进行 JSON 操作
4. **日期处理**：使用 datetime() 函数处理日期时间

## 6. 数据库管理工具

### 6.1 推荐工具

- **DB Browser for SQLite**：免费开源的 SQLite 可视化管理工具
  - 下载地址：https://sqlitebrowser.org/dl/
  - 支持查看表结构、执行SQL、导入导出数据

### 6.2 命令行工具

```bash
# 打开数据库
sqlite3 backend/data/ai_career.db

# 查看表结构
.tables

# 查看表定义
.schema users

# 执行查询
SELECT * FROM users LIMIT 10;

# 退出
.quit
```

## 7. 数据备份与恢复

### 7.1 备份

```bash
# 方法1：直接复制数据库文件
cp backend/data/ai_career.db backup/ai_career_backup.db

# 方法2：使用SQLite命令行工具
sqlite3 backend/data/ai_career.db ".backup 'backup/ai_career_backup.db'"
```

### 7.2 恢复

```bash
# 方法1：直接替换数据库文件
cp backup/ai_career_backup.db backend/data/ai_career.db

# 方法2：使用SQLite命令行工具
sqlite3 backend/data/ai_career.db ".restore 'backup/ai_career_backup.db'"
```
